- 1280x720
- 1366x768

## Checks

- Header/nav does not clip or overflow.
- Primary CTA buttons are visible without horizontal scroll.
- Cards and chat bubbles stay within viewport width.
- Dialogs remain centered and fully visible.
- Keyboard tab order reaches interactive elements.
- Focus indicators are visible for links and buttons.
